                          Python Face Detection in Images
                         ----------------------------------------
                         ----------------------------------------

This Program:
-Detects faces in images
-Prints the number of faces found
-Displays the found faces and encapsulates the faces with a blue rectangle
-Saves the new detected image containing the found faces 

In order for this program to work it requires:
-Python interpreter
-Opencv
-Matplotlib
-numpy
Python interpreter and the above libraries can be found at: 

-http://www.lfd.uci.edu/~gohlke/pythonlibs/
-https://www.python.org/downloads/

In order to run one enters the image location and the cascade path,
i.e where the haar face cascade xml file and image are.Also note that
it can also detect other facial features like eyes.
